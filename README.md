# Python BMR & BMI Calculator

Python application for calculating Basal Metabolic Rate (BMR) and Body Mass Index (BMI) using user input data. Developed with a focus on problem-solving, input validation and user-focused design. Includes software engineering documentation, testing evidence and structured program design.

![Python](https://img.shields.io/badge/Python-3.x-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## Overview

A Python application that calculates Body Mass Index (BMI) and Basal Metabolic Rate (BMR) using user input data. Developed with a focus on problem-solving, input validation and user-focused software design.

---

## Features

- BMI calculation and classification 
- BMR calculation - choice of Harris-Benedict or Mifflin-St Jeor Formulas 
- Input validation and error handling
- User-friendly interface

---

## Technologies Used

- Python
- Tkinter
- VS Code
